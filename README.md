# はじめに
このソースは、開発を終了したPraTaikoのソースです。
設計もめちゃくちゃで、改造するのも困難な状態となっています。(有識者より)

本ソースは「悪い例」として公開をしています。
これを踏み台にして、ぜひ良いものを作ってくださいね！

## リソースに関して
一部のリソースは、再配布禁止のものになっているため、本リポジトリには同梱できません。
それの影響で、起動ができない可能性がありますが、ご了承ください。

## 各ライブラリに関して
ライセンス表記をまとめるのは大変なため、お手数をおかけしますが、
各々ライブラリの導入をお願いします。
